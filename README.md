# EI-anechoic-optimization
Projet d’EI à CentraleSupélec : optimisation d’une chambre semi-anéchoïque par contrôle de la répartition d’un matériau absorbant sur la paroi. Résolution de l’équation de Helmholtz, calcul du gradient adjoint et descente de gradient pour minimiser l’énergie électromagnétique dans la zone d’intérêt.
